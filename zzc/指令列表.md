## 重磅更新！

> 使用前一定要看注意事项！！！特别说明！！！强调！！！
>
> **自造词模式开启后造词期间正常通过空格和数字123等选字**

新增自造词功能，指令输入模式如下：

### 1. 空码造词

**指令：** 空码 + `\` + 自造词 + `\`

**例如：**

```text
yzblva\要波\
```

表示这个编码是一个空码，词汇会直接补充

---

### 2. 已有词汇改词

**指令：** 词汇 + `\` + 自造词 + `\`

**例如：**

```text
yhxq\越修\
```

表示yhxq已经有词，自造词会直接顶替这个编码，原yhxq词汇会自动拆解顺延至5码

---

### 3. 指定编码长度造词

**指令：** `\` + 自造词 + 数字3-6/三-六

**例如：**

```text
\重改4
\重改四
```

表示为增加一个新词到4码位置上，如果是多音字会提供多音字全部组合供手动选择，例如重是一个多音字，所以会得到两个结果，最终手动选定

> 注意：由于暂时没有对飞键的处理，需要手动对飞键的词汇添加两次，使用更推荐1跟2的指令

---

### 4.当前编码添加额外候选

**指令：** 编码 + \ + + + 自造词 + \

**例如：**

```
ekek\+时时\
```

表示为当前编码增加一个重码候选，假设ekek只有一个候选，用此指令后，会为ekek添加一个次选

------

### 5. 删除重码2选3选等

**指令：** `\` + `-` + 数字 + `\`

**例如：**

```text
ekek\-2\
```

表示删除ekek编码的第二个候选

> 注意：此指令更适合删除有重码的编码，例如有2选3选，指定删除，不推荐直接删除只有一个词汇的编码，当然你也可以删除后立马造词补充使用

---

### 6. 置顶候选

**指令：** `\` + 数字 + `\`

**例如：**

```text
ekek\2\
```

表示把第二个候选置顶到首选

---

### 7. 词汇前移一码

**指令：** `\` + `<` + `\`

**例如：**

```text
jgwla\<\
```

表示为把jgwla这个编码的词汇前移一码，也就是5码前移至4码，4码原有词汇会自动拆解顺延至5码

---

### 8. 撤回上一次操作

**指令：** `\` + `--` + `\`

**例如：**

```text
\--\
```

表示撤回刚才的一次操作

---

## 特别强调说明！

1. 使用自造词会改动 `runtime_exact.tsv` 和写入 `zzc.dict.yaml` 文件，是你自造词记录的重要文件，不要覆盖！！！

2. `zzc.dict.yaml` 只是自造词记录文件！！！并不是真正进入你的正式码表！！！

3. 积攒一定词汇以后，请通过电脑打开 `zzc` 目录，按系统运行 `Win_词库合并.exe` / `Mac_词库合并` / `Linux_词库合并.py`，会将你的自造词自动合并到正式码表中，所以不要随便改动码表名字！！！另外，合并后会自动备份此次的 `zzc.dict.yaml` 文件，不用管
